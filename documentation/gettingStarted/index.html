Getting Started

How to get started with ZarkonUI?

Getting started with ZarkonUI isn't hard at all. It is explained in detail how to setup this library.

  1. Copy this library id: fk-LrgO0Zblzz4F10BV6Gmd3l_8mKLH-7sRe6gXz2Vg
  2. Go to the project you want to add the library in
  3. Press the gear icon (It's near the toolbox)
  4. Choose "Manage Libraries"
  5. In the input bar at the bottom of the popup, paste the library id you had copied in step one
  6. Press "Add"
  7. You have successfully setup the library! You can now use all of its functions.

Once you have this done, you should be able to use ZarkonUI's functions. To test if ZarkonUI is working, write this in the code editor:

var myButton = new ZarkonUI.Button(200, 200, 300, 80);
myButton.sprite.shapeColor = "#4287f5";
myButton.text.value = "Hello World";
var myScreen = new ZarkonUI.Screen();
myScreen.addElement(myButton);
ZarkonUI.setCurrentScreen(myScreen);
function draw() {
    ZarkonUI.manage();
}

If your result looks like this, you have done everything right! You can now start using ZarkonUI. You can read the documentations to help yourself learning.
