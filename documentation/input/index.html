Input()

Usage: new ZarkonUI.Input(x, y, width, height, id)

This function creates a button element (An object) with multiple properties, such as event and border. You can see the properties of the button object down below. Consider checking out some example(s) as well. The best part about this function is the fact that it is very difficult to create without this library, and using this library for inputs brings multiple advantages, for example it registers button presses for EVERY button. It supports shift, backspace and a lot of special characters, such as \|/{}[]:">?`~. Another advantage is the smart key hold system. For example, if you hold a key, for example "s" for more than half a second it will start typing it multiple times, like on a normal input bar.

elementType: "input" //The type of the element.
id: "<id>" //The id of the element *1
sprite: createSprite(x, y, width, height) //The sprite of the input
event: { //The events for the input
    mouseIsOver: function() {}, //Mouse is over
    mouseIsOut: function() {}, //Mouse is out
    mouseDown: function() {}, //Mouse is down
    mouseUp: function() {}, //Mouse is up
    mouseWentDown: function() {}, //Mouse went down
    mouseWentUp: function() {} //Mouse went up
}
placeholder: { //The placeholder configurations *2
    value: "", //The text on the placeholder
    color: "#b5b5b5", //The color of the placeholder
    font: "Arial", //The font of the placeholder
    size: "35", //The size of the placeholder
    outlineWeight: 0, //The weight of the outline on the placeholder
    outlineColor: "red", //The color of the outline on the placeholder
}
text: { //The text configurations *3
    value: "", //The text on the input
    color: "#000000", //The color of the text
    font: "Arial", //The font of the text
    size: "35", //The size of the text
    outlineWeight: 0, //The weight of the outline on the text
    outlineColor: "red", //The color of the outline on the text
}
border: {
    color: "#000000", //The color of the border on the input
    width: 2 //The width of the border on the input
}
cursorSettings: {
    color: "red", //The color of the cursor on the input
    width: 2 //The width of the cursor on the input
}
clone: function(newId) { /*Clones the input with a new ID*/ }
group: createGroup()  The group for the sprite of the input
draw: function() { /*Draw the input*/ }

*1: If the id wasn't specified in the 5th argument when using Input() it will be a random 1000 characters long string.

*2: The placeholder is only going to be displayed if the input isn't the active element and if there isn't any text on the input already.

*3: The text is only going to be displayed if the element is active and/or if the input has any text on it.

IMPORTANT NOTE: Before using input bars, you have to add this at the beginning of the code:

function keyTyped() {
    ZarkonUI.keyTypedSetup();
}

function keyPressed() {
    ZarkonUI.keyPressedSetup();
}

This is a very important part in using input bars. You can only put that code once, you don't have to repeat it everytime you want to use input bars.

Example 1: Creating a normal input


Code:

function keyTyped() {
  ZarkonUI.keyTypedSetup();
}
function keyPressed() {
  ZarkonUI.keyPressedSetup();
}
var input = new ZarkonUI.Input(200, 200, 300, 50);
input.border.width = 2;
input.placeholder.value = "Your name";
var screen = new ZarkonUI.Screen();
screen.addElement(input);
ZarkonUI.setCurrentScreen(screen);
function draw() {
  ZarkonUI.manage();
}
