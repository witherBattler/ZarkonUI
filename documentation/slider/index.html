Slider()

Usage: new ZarkonUI.Slider(x, y, widthThumb, heightThumb, widthTrack, heightTrack, id)

This function creates a slider element (An object) with multiple properties, such as event and border. You can see the properties of the slider object down below. Consider checking out some example(s) as well. A slider element is made to look clean in ZarkonUI. They are easy to create using it and are easily customizable. It is extremely easy to get the value of a silder, and the best part about it is the fact that it returns a value from 0 to 100 and these values are usually much easier to work with than just the difference between the slider's left edge and the thumb, because it can never go further than 100, which makes it easy to work with RGB.

{
    elementType: "slider" //The type of the element
    id: "<id>" //The id of the element *1
    track: { //Configurations related to the track of the slider
        sprite: createSprite() //The sprite of the track
        group: createGroup() //The group of the track
        event: { //The events for the track
            mouseIsOver: function() {}, //Mouse is over
            mouseIsOut: function() {}, //Mouse is out
            mouseDown: function() {}, //Mouse is down
            mouseUp: function() {}, //Mouse is up
            mouseWentDown: function() {}, //Mouse went down
            mouseWentUp: function() {}, //Mouse went up
        }
    }
    thumb: { //Configurations related to the thumb of the slider
        sprite: createSprite() //The sprite of the thumb
        group: createGroup() //The group of the thumb
        event: { //The events for the thumb
            mouseIsOver: function() {}, //Mouse is over
            mouseIsOut: function() {}, //Mouse is out
            mouseDown: function() {}, //Mouse is down
            mouseUp: function() {}, //Mouse is up
            mouseWentDown: function() {}, //Mouse went down
            mouseWentUp: function() {}, //Mouse went up
        }
    }
    draw: function(newId) { /*Draw the slider*/ } 
    getValue: function() { /*Get the value from the slider (from 0 to 100)*/ } 
    clone: function() { /*Clone the slider with a new ID*/ }
}

*1: If the id wasn't specified in the 5th argument when using ZarkonUI.Slider() it will be undefined and you will run into issues while trying to use getElementById().
