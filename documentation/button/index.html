Button()

Usage: new ZarkonUI.Button(x, y, width, height, id)

This function creates a button element (An object) with multiple properties, such as event and border. You can see the properties of the button object down below. Consider checking out some example(s) as well.

{
    elementType: "button" //The type of the element.
    id: "<id>" //The id of the element *1
    sprite: createSprite(x, y, width, height) //The sprite of the button
    event: { //The events for the button
        mouseIsOver: function() {}, //Mouse is over
        mouseIsOut: function() {}, //Mouse is out
        mouseDown: function() {}, //Mouse is down
        mouseUp: function() {}, //Mouse is up
        mouseWentDown: function() {}, //Mouse went down
        mouseWentUp: function() {} //Mouse went up
    }
    text: { //The text configurations
        value: "", //The text that is going to be shown on the button
        color: "#000000", //The color of the text
        font: "Arial", //The font of the text
        size: "35", //The size of the text
        outlineWeight: 0, //The weight of the outline on the text
        outlineColor: "red", //The color of the outline on the text
    };
    border: {
        color: #000000 //The color of the border
        width: 0 //The width of the border
    }
    clone: function(newId) { /*Clones the button with a new ID*/ }
    group: createGroup()  The group for the sprite of the button
    draw: function() { /*Draw the button*/ }
}

*1: If the id wasn't specified in the 5th argument when using Button() it will be a random 1000 characters long string.

Examples

Example 1: Creating a normal button


Code:

var myButton = new ZarkonUI.Button(200, 200, 300, 80);
myButton.sprite.shapeColor = "#4287f5";
myButton.text.value = "Hello World";
myButton.event.mouseIsOver = function() {
  myButton.sprite.shapeColor = "red";
};
myButton.event.mouseIsOut = function() {
  myButton.sprite.shapeColor = "#4287f5";
};
var myScreen = new ZarkonUI.Screen();
myScreen.addElement(myButton);
ZarkonUI.setCurrentScreen(myScreen);
function draw() {
  ZarkonUI.manage();
}
